# Transport Management System

A full-stack **MERN** (MongoDB, Express, React, Node.js) web application designed for managing transport operations efficiently.

## Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Atlas)

##  Features
- Manage Drivers, Vehicles, and Routes
- Track Deliveries and Status Updates
- Role-based Login (Admin, Staff)
- Real-time Data via MongoDB Atlas
